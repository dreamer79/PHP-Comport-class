PHP-Comport-class by dreamer79

PHP PECL extension for comunicating with com(serial) ports on Linux/Windows

PHP Comport class is a project for creating a simple layer for serial comunication in PHP with external devices- modems, cash registers, scales, label printers... The extension is avaible both for Windows and Linux/POSIX systems and is an alternative to the phpSerial class that can READ serial ports on Win32 systems.

Dependencies

PHP source, Visual C++ compiler on Windows systems

Install

See README for details how to compile. Has a compiled .dll file for Windows PHP5.3.8TS
